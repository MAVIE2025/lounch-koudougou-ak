# LOUNCH KOUDOUGOU AK - Railway

## Déploiement Railway

1. Créer un dépôt GitHub.
2. Envoyer ces fichiers.
3. Aller sur Railway.
4. New Project -> Deploy from GitHub repo.
5. Ajouter PostgreSQL dans Railway.
6. Railway crée automatiquement DATABASE_URL.
7. Déployer.

Compte par défaut :
- utilisateur : admin
- mot de passe : admin123

## Important
Cette version utilise PostgreSQL. Elle est mieux adaptée à Railway que SQLite, car Railway peut redéployer l'application.
